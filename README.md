# zhs-cn-notes

## 项目简介

zhs-cn-notes 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面可能涵盖笔记、文档、工具或其他静态内容，不针对任何特定网站或域名。本仓库旨在提供一个整洁、可访问的存储空间，方便浏览和引用。

## 目录说明

- `pages/` —— 存放主要的 HTML 页面文件。每个页面通常独立成文，内容自包含。
- `assets/` —— 页面所需的资源文件，如图片、样式表、脚本等（如有需要）。
- `index.html` —— 仓库首页，提供页面列表或导航入口（可选）。
- `README.md` —— 本文件，仓库说明。

## 页面归档说明

- 每个 HTML 页面均为独立归档，不依赖外部数据库或后端服务。
- 页面内容可能经过整理、摘录或翻译，但保留原始信息的核心。
- 部分页面可能包含时间戳或版本标记，以体现归档顺序或更新情况。
- 页面之间的链接保持相对路径，便于本地查看或部署到静态托管服务。
